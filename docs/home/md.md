## 代码块中的特殊行

```javascript
var a = 10; // [!code hl]
var a = 11;
var a = 12; // [!code hl:2]
var a = 13;
var a = 14; // [!code ++]
var a = 15; // [!code --]
var a = 16; // [!code focus]
var a = 17; // [!code error]
var a = 18; // [!code warning]
```

````markdown
```javascript
var a = 10; // [!code  hl]
var a = 11;
var a = 12; // [!code  hl:2]
var a = 13;
var a = 14; // [!code  ++]
var a = 15; // [!code  --]
var a = 16; // [!code  focus]
var a = 17; // [!code  error]
var a = 18; // [!code  warning]
```
````

> // [!code xxx:lines] 代码高亮语法

- xxx **取值有**
  - `hl` 高亮
  - `++` 表示添加
  - `--` 表示减少
  - `focus` 表述聚焦
  - `error` 表述错误
  - `warning` 表示警告
- lines **表示往下应用的行数**
- `code` 和 `xxx` 中间只能有一个空格才有效
