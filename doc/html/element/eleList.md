# 列表元素

## 有序列表

ol: ordered list

li：list item

<ol>
  <li>打开冰箱门</li>
  <li>大象进去</li>
  <li>冰箱门关上</li>
</ol>

```html
<ol>
  <li>打开冰箱门</li>
  <li>大象进去</li>
  <li>冰箱门关上</li>
</ol>
```

## 无序列表

把 ol 改成 ul

ul：unordered list

> 无序列表常用于制作菜单 或 新闻列表。

xxx 美女择偶条件：

<ul>
    <li>有责任心</li>
    <li>会家务</li>
    <li>孝敬老人</li>
    <li>年薪30W</li>
</ul>

```html
xxx美女择偶条件：
<ul>
  <li>有责任心</li>
  <li>会家务</li>
  <li>孝敬老人</li>
  <li>年薪30W</li>
</ul>
```

## 定义列表

> 通常用于一些术语的定义

dl: definition list

dt: definition title

dd: definition description

HTML 中的术语解释：

<dl>
    <dt>HTML</dt>
    <dd>
        超文本标记语言，XXXXXX
    </dd>
    <dt>元素</dt>
    <dd>
        组成HTML文档的单元，每个xxxxx
    </dd>
</dl>

```html
HTML 中的术语解释：

<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言，XXXXXX</dd>
  <dt>元素</dt>
  <dd>组成HTML文档的单元，每个xxxxx</dd>
</dl>
```
